# mlhelper
R package for EDA and ml preprocessing

Installation steps

Install from github:
install_github("szintakacseva/mlhelper")

Load library:
library(mlhelper)

Optionally load from local repository:
library(mlhelper, lib.loc = "c:/Yor_local_dir")
for example,
library(mlhelper, lib.loc = "e:/RProjects/packages")
